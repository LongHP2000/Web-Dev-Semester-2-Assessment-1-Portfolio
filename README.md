# Web Dev I - Set Exercises Assessment Portfolio
# By Long Pham

This repository contains my solutions for the Set Exercises assessment for the Web Dev I (CCO4007-20P,SEP-CA,TRI1,2025-2026) module at Bath Spa University.

## Overview

This assessment tests my knowledge of fundamental web development techniques using HTML and CSS. It consists of four exercises, each focusing on a different area of web development.

## Exercises

This repository is organized into separate folders for each exercise:

*   **Exercise 1: Semantic Markup** - Focuses on using semantic HTML elements to structure content.
*   **Exercise 2: CSS Styling** - Focuses on styling web pages using CSS.
*   **Exercise 3: Page Layout** - Focuses on creating effective page layouts using CSS.
*   **Exercise 4: Web Accessibility** - Focuses on creating accessible web content.

Each folder contains the necessary files for the exercise, as provided in the assignment.
